## About The Project

<!-- [![Product Name Screen Shot][product-screenshot]](https://example.com) -->

Diseño y desarrollo de una página educativa para diferentes rangos etarios, se realizará con las tecnologías dadas por la capacitación de educación IT.
La página web contendra un inicio donde se mostraran los cursos, filtrado de cursos, varias secciones como Inicio, foro, contacto y nosotros. Un login, una página de registro y varios estilos dependiendo de la edad.


Contenido del proyecto:
* Contendra un inicio con nuestro logo y objetivo.
* Sección de cursos, con filtrado.
* Un foro para publicar todas las dudas y/o soluciones.
* Secciones con nuestro contacto y datos sobre nosotros.


Queremos evolucionar los e-learning dando muchos cursos con temáticas que abarquen varios rangos etarios.

## Getting Started

Aquí van a ir los pasos para poder montar y ejecutar el proyecto.

### Prerequisites

Programas requeridos:
* XAMPP
  ```sh
    En el proceso de instalación dejen las casillas marcadas solo MySQL y phpmyadmin, el resto dejenla desmarcada
  ```

### Installation

Pasos para inicializar el proyecto:

1. Clonar el repo
   ```sh
   git clone https://github.com/BOTPili/proyecto-pescar.git
   ```
2. Ahora ponga el proyecto donde se instalo XAMPP en la carpeta => \htdocs
3. Para ver el proyecto online abran XAMPP y activa donde dice "Apache" y ahi les corre el servidor
4. Vayan a su navegador y en la url ponen "localhost nombre-del-proyecto"
